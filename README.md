# Advent of Code

[https://adventofcode.com/](https://adventofcode.com/)

## Setup

1. ```bash
   python3 -m venv .venv
   ```

2. ```bash
   source .venv/bin/activate
   ```

3. ```bash
   pip install --upgrade pip
   ```

4. ```bash
   pip install -r requirements.txt
   ```

5. ```bash
   pip install -e ./utils
   ```
